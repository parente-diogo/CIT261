JSON

A common use for JSON is sending and retrieving data to and from a web server.

Data in a web server is only stored in string format.
Prior to sendind any data to a web server, it is needed to convert it to string using Stringify().
After receiving any data from a webserver, it is needed to convert it back from string to a JavaScript Object using Parse().
