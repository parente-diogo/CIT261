JSON

A common use for JSON is sending and retrieving data to and from a web server.

Data in a web server is only stored in string format.
Prior to sendind any data to a web server, it is needed to convert it to string using Stringify().
After receiving any data from a webserver, it is needed to convert it back from string to a JavaScript Object using Parse().

Parse

As long as the information to be retrieved is in JSON format it is possible to parse it back from string to an JavaScript object.
If the information to be retrieved is not in JSON format there will be an error in the webpage when running the parse method.

The first button will retrieve the information from the Local Storage but not parse it. There will be no parse error in the webpage, even when the information is not in JSON format, but the original objects will return as strings. The length of characters will count even the brackets and commas.
The second button will retrieve the information from the Local Storage and parse it. If the information is in JSON format, they will return as JavaScript objects.
