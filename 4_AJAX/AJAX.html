AJAX

AJAX makes possible to:

  • Read data from a web server - after the page has loaded;
  • Update a web page without reloading the page;
  • Send data to a web server - in the background.

JSON

JSON stands for JavaScript Object Notation.

JSON is a syntax for sending to and requesting data from a server. Exchange data between the browser and the server.

JSON is text only. It is needed to use the commands JSON.stringify() and JSON.parse().
