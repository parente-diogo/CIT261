LOOPS

A Loop is a block of code made to be repeated. It runs the same code, but can use different values each time to achieve different results.

-----------

For Loop

The For Loop will run the code as much times as specified by the values in the statements and/or inside the loop.
It can have 3 statements. All three statements are optional.

Statement 1 Initialization of variables. Any amount of variables.
Usually the variable used in the loop is initiated.
Statement 2 Evaluate the condition to run the loop. When the condition is TRUE the loop will run.
Must have a condition that will turn to FALSE sometime or have a break. Otherwise the loop will never end and the browser will stop working.
Statement 3 Usually used to increment or decrement the loop counter. It is possible to use it for any other logic.
The loop counter can be changed inside the loop.

While / Do While Loop

The While Loop will run the code WHILE the assessed condition is true.

The Do While Loop will run the code at least once, even if the condition is false. After each time the code is run, the condition is evaluated to run it again or not.

CONDITIONAL STATEMENTS

If Execute the code when the actual condition is true. It is not affected by any previous statement conditions.
Else if Execute the code ONLY when the last condition was false, and the actual condition is true.
Else Execute some code when the last condition was false. It will not evaluate any condition.
Switch Specify many blocks of code, equal or different. Each will be performed based on different conditions.

FUNCTIONS

A Function is a block of code created to execute a task. The function will be executed when something calls it.

function Keyword used to define the following code, inside curly brackets, as a function.
Name Identification to the block of code to be called as a function. Each function must have a different name.
Parameters Variables declared while creating the function. It is the place to insert values to be used by the function. Placed between parentheses ( ... ).
Arguments Values inside the function to be used by the code.
Return Statement used inside the function to output a final value to the function's caller. It also stop the execution of the block of code.

The variables declared inside a function stay inside the function, they have no effect outside that function, they are Local Variables.

VARIABLES

Variables are containers for storing data values.

The declaration of a variable is possible using three different statements:

var Create a Global variable possible to be changed/reassigned in the future.
let Create a Local variable possible to be changed/reassigned in the future.
const Create a Local variable; its assigned value is constant being not allowed to be changed/reassigned in the future.

ARRAYS

An array is a special object that can hold many values. These values can be of the same type or different types.

When using the typeof operator in an array, it will return "object" as the result and not "array".

To access one element of an array, it is used the index of the element inside that array. The index is the place where it stays.
The indexes start in zero.
myarray[2] will access the third element inside the array called "myarray".

Array

An Array should be used when the element names (indexes) are required to be numbers.

fruits = ["Banana", "Orange", "Apple", "Strawberry"]

Associative Array

In JavaScript, Associative Arrays are called Objects.

An Object should be used when the element names (indexes) are required to be strings.

person = { "name":"John", "Surname":"Doe", "age":40, "alive":TRUE }
