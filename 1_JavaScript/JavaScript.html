JAVASCRIPT

LOOPS

A Loop is a block of code made to be repeated. It runs the same code, but can use different values each time to achieve different results.

-----------

For Loop

The For Loop will run the code as much times as specified by the values in the statements and/or inside the loop.
It can have 3 statements. All three statements are optional.

Statement 1 Initialization of variables. Any amount of variables.
Usually the variable used in the loop is initiated.
Statement 2 Evaluate the condition to run the loop. When the condition is TRUE the loop will run.
Must have a condition that will turn to FALSE sometime or have a break. Otherwise the loop will never end and the browser will stop working.
Statement 3 Usually used to increment or decrement the loop counter. It is possible to use it for any other logic.
The loop counter can be changed inside the loop.

While / Do While Loop

The While Loop will run the code WHILE the assessed condition is true.

The Do While Loop will run the code at least once, even if the condition is false. After each time the code is run, the condition is evaluated to run it again or not.

CONDITIONAL STATEMENTS

If Execute the code when the actual condition is true. It is not affected by any previous statement conditions.
Else if Execute the code ONLY when the last condition was false, and the actual condition is true.
Else Execute some code when the last condition was false. It will not evaluate any condition.
Switch Specify many blocks of code. Each one, some or none blocks can be run depending on the conditions.
