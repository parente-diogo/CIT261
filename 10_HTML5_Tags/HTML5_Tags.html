HTML5 Tags

VIDEO

source: path to the location of the video. The browser will use the first recognized format; it is possible to have more than one source;
width: fixed horizontal size for the video screen;
height: fixed vertical size for the video screen;
type: Three formats are supported - "video/mp4", "video/webm" and "video/ogg";
controls: control buttons for the video (play, pause, time counter, slide bar, volume, other options);
autoload: starts the video automatically.

AUDIO

source: path to the location of the audio. The browser will use the first recognized format; it is possible to have more than one source;
type: Three formats are supported - "audio/mpeg", "audio/wav" and "audio/ogg";
controls: control buttons for the audio (play, pause, time counter, slide bar, volume, other options).
