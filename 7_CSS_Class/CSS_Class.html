Manipulating CSS Properties

This current webpage has no CSS stylesheet to style colors. All styles are made using JavaScript functions to modify properties:

  1. Using a script in the HTML file;
  2. By a function in the buttons (only ID elements);
  3. By a JavaScript function loaded by the buttons.

ID Attributes

To change attributes calling its ID Name it is possible to change it directly in the button using onclick, it is not needed to run a For Loop Iteration.
For each page it is possible to have only one element specified with each ID name. Then getElementById returns only one element, and not a collection.

Class Attributes

To change attributes calling its Class Name it is needed to used a JavaScript function, and run a For Loop Iteration.
The getElementsByClassName() method returns a collection of all elements in the document with the specified class name. It is possible to have many elements specified with the same class name.
