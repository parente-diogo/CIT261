DOM

DOM stands for Document Object Model.
It is a tree containing every object in the page, connected according to the relationship to other elements, descendants, siblings, or whatever relation they have.
The browser creates one DOM for each page loaded.

Knowing the DOM for a webpage makes it possible to change its HTML in the JavaScript code.
