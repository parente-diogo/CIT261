Variables

A String is a type variable that holds one only value in the format of text.
Strings have Properties that are characteristics of the value inside. One property is .length that shows how many characters the value has.
Strings also have Methods that are like functions that accomplish particular acts. One method is .toUpperCase() used to make all letters uppercase.

JavaScript Objects

Definition from geeksforgeeks.org: "... objects in JavaScript may be defined as an unordered collection of related data, in the form of “key: value” pairs. These keys can be variables or functions and are called properties and methods, respectively".

An Object has similarities compared to a Variable. An object can have properties, not only one, but many properties. Can exist many methods for an object, all the methods can be applied to each property.

Example:
A Computer can be an Object.
It can have some properties like Name, MemorySpace, ScreenSize, Processor.
Each property can have one value.
Possible methods are .TurnON(), .TurnOFF(), .FormatC(), .RunVideo().
