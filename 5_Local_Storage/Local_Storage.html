API

Application Programming Interface

It is an intermediary application that links a main service to an user.
The user can be another service, website, machine.

Example: Google Maps is the main service of maps and localization from Google. The user can be a website of a company located anywhere in the World. The API that access information from Google Maps can be installed in this company site to show its real location. The company website does not have any map, but it can show the map because the API links it to the main Map Service.

Web Storage

Session Storage

Stores data until the session is closed. When you close the tab or the browser the information is erased.

Local Storage

Maintains the information even after closing the browser, there is no expiration date. To erase the information you need to clean the browser cache
