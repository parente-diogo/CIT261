CSS3 Transitions and Animations

TRANSITIONS

A Transition is a gradual change of value to a property during a set time, making the change smooth.

transition-delay: Time to start the transition.
transition-duration: Time the transition will need to reach its ending.
transition-property: Choose which property is under transition for the element.
transition-timing-function: Behavior (speed curve) for the transition to obey.
transition: shorthand to include all transition properties in the same line.

Trigger: Hover
Delay: 0 seconds
Duration: 1 second
Property: width
Timing function: default
