JavaScript Events

An event is a "thing" that happens to a HTML element and trigger another action. JavaScript functions can "react" to these triggers and perform something.
Event is a way of interaction between JavaScript and HTML codes.

Touch Events

Attribute Description
ontouchcancel The event occurs when the touch is interrupted.
ontouchend The event occurs when a finger is removed from a touch screen.
ontouchmove The event occurs when a finger is dragged across the screen.
ontouchstart The event occurs when a finger is placed on a touch screen.

Animation Events

Attribute Description Action Example
animationstart The event occurs when a CSS animation is completed.
animationend The event occurs when a CSS animation is repeated.
animationiteration The event occurs when a CSS animation has started.

Transition Events

Attribute Description
transitionend The event occurs when a CSS transition has completed.
